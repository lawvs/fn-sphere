## Installation

import { Steps } from "@astrojs/starlight/components";
import { Tabs, TabItem } from "@astrojs/starlight/components";
import { Code } from "@astrojs/starlight/components";

<Steps>

1. Install the Filter Sphere package using your favorite package manager:

   <Tabs syncKey="package-manager">
     <TabItem label="pnpm">
       <Code lang="bash" code="pnpm add @fn-sphere/filter" />
     </TabItem>
     <TabItem label="npm">
       <Code lang="bash" code="npm install @fn-sphere/filter" />
     </TabItem>
     <TabItem label="yarn">
       <Code lang="bash" code="yarn add @fn-sphere/filter" />
     </TabItem>
   </Tabs>

1. Filter Sphere uses [zod](https://zod.dev/) to define your data schema. If not installed, run the following command:

   <Tabs syncKey="package-manager">
     <TabItem label="pnpm">
       <Code lang="bash" code="pnpm add zod" />
     </TabItem>
     <TabItem label="npm">
       <Code lang="bash" code="npm install zod" />
     </TabItem>
     <TabItem label="yarn">
       <Code lang="bash" code="yarn add zod" />
     </TabItem>
   </Tabs>

1. Start building your filtering interface with Filter Sphere components.

</Steps>
